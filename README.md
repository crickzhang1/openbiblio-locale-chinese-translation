openbiblio-locale-chinese-translation
=====================================

(author: crickzhang1)

Chinese translation for OpenBiblio i18n. For now the work is for
the latest release (0.7.1) of OpenBiblio.

## Status

* Currently only translation for simplified Chinese is ongoing.
